What is RSA SecurID?

RSA SecurID ("SecurID") is a two-factor authentication solution that is used to protect network resources. The authentication is based on two factors â€” something you know (a PIN) and something you have (a token). The SecurID solution facilitates consistent and secure Multi-Factor Authentication (MFA) for users on the COE network.

Benefits

  • The RSA SecurID Token authentication solution facilitates consistent and secure two-factor authentication.
  • The RSA SecurID Token provides a critical network login enhancement and improved security.
  • Adoption of the MFA enabled by use of the RSA SecurID Token helps make the Top Secret network more secure.
